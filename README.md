# 🚀 Flask User Dashboard (Microservice Project)

A simple Flask-based **User Management Microservice** for performing CRUD operations.

---

## 🌐 Live Demo
👉 [https://arpita2005.pythonanywhere.com](https://arpita2005.pythonanywhere.com)

---

## 📌 About Project

This is a **Flask microservice application** that allows you to:

- 🔍 Search user by ID  
- ➕ Add new user  
- ❌ Delete user  
- 📋 View all users  

It follows a modular backend structure using routes, services, and models.
